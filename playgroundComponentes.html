Encabezado de Secciones (headerWrapper)

Este cosito se usa para encapsular titulos que permiten distinguir secciones dentro de la pagina. Puede usarse con etiquetas H1, H2 y H3. El tamaño del elemento cambiara segun el nivel de titulo usado. La combinacion de colores puede modificarse directamente en el atributo style.

Tarjetas Horizontales (horizontalCard)

Este componente se utiliza para presentar informacion en tarjetas horizontales. Se pueden cambiar los colores con el atributo style.

Tarjetas verticales (VerticalCard)

Galeria (Instancia de PhotoSwipe)

Galeria de imagenes con lightbox. Para funcionar, este componente utiliza PhotoSwipe V5. Para llamar el paquete sin necesidad de crear un build usamos UNPKG.

Notas importantes:

  • Es necesario incluir el link de la imagen en dos elementos (a y en img)
  • hay que definir el tamaño de la imagen una vez que este en el lightbox (que pereza haha) Seguramente hay opciones para cambiar este comportamiento. Ver documentacion.
  • el caption que se muestra es opcional (eleminar elemento p)
  • Es posible agregar captions al lightbox. ver plugin.
